🎯 Sobre o ClassCheck

Sistema web para avaliação educacional que permite alunos avaliarem aulas e professores, registrarem humor diário e receberem feedback. Professores acessam relatórios detalhados e a diretoria visualiza métricas consolidadas através de dashboards interativos.

PWA Ready Responsive Design Real-time Analytics Multi-role System
